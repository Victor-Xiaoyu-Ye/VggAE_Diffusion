# Required HuggingFace Models

These will be auto-downloaded on first use, or pre-download them:

## 1. Wan2.1 1.3B (Diffusion Backbone)

```bash
huggingface-cli download Wan-AI/Wan2.1-T2V-1.3B \
  --local-dir Wan2.1/checkpoints/Wan2.1-T2V-1.3B
```

Used by: `models/wan_adapter.py`

## 2. CLIP ViT-L/14 (Text Encoder)

```bash
# Auto-downloaded by transformers on first use
# Cache: ~1.5 GB
```

Model: `openai/clip-vit-large-patch14`

Used by: `models/clip_encoder.py`

## 3. LPIPS VGG Weights

Auto-downloaded by the `lpips` package on first use (~50 MB).
